# Drone Simulink Model Setup

## Required Software

* MatLab Software
  * **[UAV Toolbox Support Package for ArduPilot Autopilots](https://www.mathworks.com/help/releases/R2026a/uav/ardupilot/ug/install-ardupilot-autopilot.html)**
    * This software is used to directly interface with mission planner using the simulink model
  * [Embedded Coder](https://www.mathworks.com/help/ecoder/index.html)
  * [MATLAB Coder](https://www.mathworks.com/help/coder/index.html)
  * [Simulink](https://www.mathworks.com/help/simulink/index.html)
  * [UAV Toolbox](https://www.mathworks.com/help/uav/index.html)
  * [Simulink Coder](https://www.mathworks.com/help/rtw/index.html)
  * [Aerospace Blockset](https://www.mathworks.com/help/aeroblks/index.html)
  * [Aerospace Toolbox](https://www.mathworks.com/help/aerotbx/index.html)
  * [Instrument Control Toolbox](https://www.mathworks.com/products/instrument.html)

* Missionplanner
  * [Missionplanner Installation Guide](https://ardupilot.org/planner/docs/mission-planner-installation.html)
 
* Ardupilot Build Setup
  * [Setting Up the Build Environment](https://ardupilot.org/dev/docs/building-setup-linux.html)
